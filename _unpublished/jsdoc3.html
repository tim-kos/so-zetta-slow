Comecei a brincar com esta ferramenta para documentação inline de código JavaScript e estou gostando. Pretendo criar a documentação da minha calculadora de dano para Pokémon com ela.

Para usá-la, deve ter o Java 1.7 instalado e configurado no seu computador. Depois, é só baixar do repositório no Github:

git clone https://github.com/micmath/jsdoc.git cd jsdoc

Agora é uma boa rodar os testes para verificar que está tudo ok.

./jsdoc -T

Finalmente, use a documentação para saber quais tags são reconhecidas e como modificar o tema básico. Um pequeno exemplo é o seguinte:

/** @version 2.0 @license MIT */

Quando compilado (com ./jsdoc path/to/javascript-source.js), vai criar uma pasta out/ com um arquivo HTML.

Apesar de ainda estar em fase beta, é bem simples de usar e parece que 'vai para a frente'. Quando eu tiver algo feito com isso, colocarei aqui.
